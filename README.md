# SWIM (Shear Wave Interference Modulation - 剪切波干涉调制)

Title: `Spatiotemporal Interference of Cutaneous Shear Waves Dictates Tactile Perception in Ultrasonic Haptics`

## 研究目标

研究目标重点并不是单纯简单的提出一种新的调制方式，而是通过实验、仿真和理论分析，来深入的研究在不同调制模式下，效果的差异的根本原因/机理（心理学原因，生理物理学原因等等）。偏底层基础研究（如剪切波传播、马赫锥效应、机械感受器激活），并给出一套新的从表面波传播角度分析和看待的超声触觉形成相关的理论（虽然相关研究已经表明了皮肤表明剪切波传播才是产生触觉更强的主要因素，并且也有研究研究了皮肤表面的表面波的马赫锥会导致触觉扩散模糊，但是仍然缺少一个统一的来指导超声触觉设计的理论）。以及给出从实验结果和形成的理论上分析，想要实现强度更高、精度更高的超声触觉反馈，需要满足的条件。

### 目标期刊

Nature/Science 旗下的所有适合的子刊。

### 论文包装叙事线

最优中心思想（核心叙事主线）：超声触觉感知的决定因素并非局部的物理极值（能量、梯度、应变率），而是剪切波前在皮肤表面的时空连贯性（Spatiotemporal Coherence）与受体敏感频段的严格对齐（Frequency Fidelity）。

破题：指出当前超声触觉设计的盲区——过度追求焦点在空间上的物理极值，忽略了表面波的动态传播与感受器时间频率响应的匹配。
立论：通过 ULM_L 的“低物理极值-高感知强度”以及 LM_L 的“绝对倍频毁灭感知”现象，打破传统直觉。
建构：提出“相干积分（Coherent Integration）”理论。由于组织衰减极大，唯一的解法是利用马赫锥效应（波速匹配）进行单向、无破坏性干涉的持续波前压缩，这能保证机械能在受体最佳频带（200Hz）内实现最大化的有效神经积分。

## 背景信息

### 问题分析

传统的LM调制，虽然通过横向移动超声相控阵焦点，实现触发更多的皮肤受体，结合对焦点移动速度的控制，使得焦点移动速度匹配皮肤表面剪切波波速，从而实现对波前的压缩。通过这两个效应达到了更强的触觉反馈强度。但是这种方法存在触觉反馈精度的问题，焦点超音速（皮肤表面波音速）移动时，会产生马赫锥效应，在皮肤表面扩散开来，导致触觉生成精度差。

### 调制方式设计

我现在有一种离散的超声触觉焦点调制方法，名称为 Discrete Lateral Modulation（DLM）。每个时刻焦点都不进行移动，而是在多个不同位置之间进行瞬间切换。

具体细节为：超声相控阵的聚焦点周期性的在一个圆环上均匀分布的多个离散点之间进行切换，每个周期（例如30Hz的一个周期）内超声聚焦点按顺序在每个聚焦点之间转移，例如我上传的两个图片示意了两个焦点切换和三个焦点切换。当采样的离散点个数趋于无穷个时，这种焦点调制方法等效于圆形轨迹的横向调制（LM_C）。

我还根据LM的论文设计了一种 **LM_L** 的变体版本 **ULM_L** (Unidirectional Lateral Modulation (Linear))，聚焦点不是在两个目标点之间来回移动，而是单向的从A点移动到B点，再瞬移到A点继续直线移动到B点。这种调制方式保证了轨迹上每个点受到的刺激频率和轨迹的循环频率保持一致（都是200Hz），不会出现双向的 **LM_L** 由于每个周期经过轨迹中间每个点两次从而产生的2倍频到400Hz的现象。从而保证了最大程度刺激皮肤受体敏感的频率区间。（可能非原创，其他人应该已经做过了，但是可以用来进行对比试验，作为探针来分析数据和理论）。

### 理论分析

我自己对 DLM、ULM_L、LM_L 和 LM_C 调制的初步分析如下：

- 两点DLM（DLM_2）：当我切换聚焦点的时机足够恰好时，在第一个聚焦点A戳下去，等产生的皮肤表面传播的剪切波刚好传播到B处时，再切换焦点到B点。这样会在B点形成一个很大的压力或者剪切梯度，从而更高效的激活皮肤触觉受体，产生明显的触觉感知。

- 三点DLM（DLM_3）：当离散的聚焦点个数增加到3个及以上时，在DLM_2的基础上，每个焦点产生的表面波还会在圆心处产生一个旋转变化的剪切应变/应力，这个旋转剪切量旋转的频率正好是调制整体调制频率（200Hz），从而同样可以激活皮肤中对剪切敏感的触觉受体）。

- ULM_L：ULM_L 调制方式的每个周期内，焦点会单向的直线从A点移动到B点移动，焦点移动速度匹配皮肤表面剪切波波速，从而实现对波前的持续压缩。

- LM_L：持续移动的 **LM_L** 调制方式的焦点会在焦点移动方向前侧产生一个高压区，在焦点移动方向后侧产生一个低压区。当焦点移动到直线线段的两侧两头时，速度方向突然180° 转变，则会让新产生的高压区和之前尾迹的低压区碰撞，产生一个极大的压力或者剪切的梯度，从而同样产生一种很强大的触觉感知。

- LM_C：平滑沿着圆形轨迹移动的焦点在每一时刻都会在皮肤上激发沿着皮肤方向传播的表面剪切波，和DLM类似，这些剪切波也会在圆心处产生旋转剪切场。


### 超声触觉设备 UMH Device 信息

我们研究开发了一种高能量密度，小尺寸的集成式超声触觉设备 UMH Device（Ultrasound Mid-Air Haptic Device），本研究中所进行的超声触觉实验均是在这台设备的基础上进行的。以下是 UMH 设备的详细结构化描述。

#### 1. 设备物理与形态规格

- **整体外形**：正六边形设计，对边距 90mm。

- **三维尺寸**：最大外形尺寸为 102.2 × 90 × 32.2 mm（包含换能器、PCB、铜柱及接口突出部分）。
- **阵元排布**：
  - 采用六边形最密堆积（Hexagonal Close Packing）。
  - 边长为 5 个阵元，理论满排布为 61 个阵元。
  - 实际排布移除最中心位置（未来预留放置 CMOS 图像传感器），实际搭载 **60个** 超声换能器。

#### 2. 核心硬件与主控参数

- **微控制器 (MCU)**：STM32H750VBT6。
  - 运行频率机制：系统使用 DWT 进行周期计数和微秒级延时，通过 SysTick 和内建定时器（TIM1）同步 DMA 触发。

- **供电系统**：
  - 主输入电压：12V DC。
  - 内部 ADC 监测电路（通过 ADC1 和 ADC3）：实时监测 `3.3V`、`5.0V` 电压轨以及 `VDDA` 和 MCU 内部温度。

#### 3. 超声换能器 (Transducer) 规格

- **型号**：GU1007C-40TR（铝壳开放式空气超声换能器，收发一体）DONGUANINGHAI ELECTRONICS CO.,LTD.

- **物理尺寸**：直径 10mm，高度 7mm。
- **声学参数**：
  - 中心工作频率：40 kHz。
  - 波长 ($\lambda$)：在室温（声速 $343.2\text{ m/s}$）下约为 8.58 mm。
  - 阵元间距 (Transducer Spacing)：10 mm。
  - 指向性 (Directivity)：约 80°。

#### 4. 驱动电路与波形生成 (Bipolar Drive)

- **驱动芯片**：IX4428MTR（双通道高速低边 MOSFET 驱动器）。IXYS Integrated Circuits Division

- **驱动拓扑**：
  - 输入单路信号 (`SIG`)，通过电阻与 ESD 保护二极管进入驱动 IC 的 `INA`，同时通过外围反相网络（或芯片内部逻辑）使 `INB` 获得反相信号。
  - 输出端 `OUTA` 和 `OUTB` 接换能器两端，形成**双极性驱动（Bipolar Drive）**或全桥推挽效果。
  - **驱动电压**：驱动 IC 由 12V (`VDD`) 供电。利用推挽输出，换能器两端峰峰值电压 (VPP) 翻倍，达到 **24 VPP**。

#### 5. 波形输出与 DMA 机制

- **相控阵控制方式**：通过 STM32 的 Timer + DMA（Mem-to-Mem 模式映射到 GPIO ODR 寄存器）实现高精度多通道同步输出。

- **DMA 通道分布**：使用 5 个 DMA 数据流（`DMA1_Stream0~2`, `DMA2_Stream0~1`），分别控制 5 个 GPIO 端口 (`GPIOA` 到 `GPIOE`)。
- **采样率与分辨率**：
  - DMA 采样频率设定为 4 MHz (`DMA_SAMPLING_FREQ`)。
  - 对于 40kHz 的载波，每个超声波周期在缓冲区中被划分为 100 个时间片（Buffer Size = 100）。
  - 相位延迟通过换算为缓冲区的偏移位（`shift_buffer_bits`）来实现，具备纳秒级的相位控制精度。

#### 6. 固件算法与声场控制 (Stimulation)

设备通过计算换能器三维坐标到目标焦点的空间距离差异，实现波束成形（Beamforming）。支持多种声场控制模式（定义在 `stimulation.c` 中）：

- **Focus Point (单点聚焦)**：将声能汇聚于三维空间的一点。
- **Discrete / Linear / Circular 轨迹**：焦点在空间中沿离散点、直线或圆形轨迹进行周期性扫描/移动。
- **TwinTrap (双声阱)**：生成用于声悬浮的 TwinTrap 拓扑（通过几何分割，一半换能器施加额外的 $\pi$ 相位差）。
- **相位刷新率**：控制算法以设定的刺激频率（如 200 Hz，即周期为 5ms）在缓冲区内动态更新各个阵元的相位状态。

#### 7. 通讯协议与控制接口

- **物理接口**：USB Type-C。

- **通讯模式**：USB CDC（虚拟串口）。
- **通信帧结构**：
  - 定制的二进制帧协议，最大帧长 260 字节。
  - **格式**：`[Header1 0xAA] [Header2 0x55] [CMD Type] [Data Length] [Data...] [Checksum] [Tail1 0x0D] [Tail2 0x0A]`。
- **支持的控制命令 (CMD)**：
  - `0x01`：启用/禁用刺激输出。
  - `0x02`：Ping 联机测试。
  - `0x03`：获取设备配置（序列号、版本、阵元数、间距等）。
  - `0x04`：获取设备状态（电压、温度、循环频率、当前模式等）。
  - `0x05`：下发并设置声场控制参数（类型、位置、法向量、半径、频率等）。
  - `0x06`：直接设置底层所有阵元的绝对相位。
  - `0x07`：切换内置的演示模式 (Demo Mode)。
  - `0x08`：获取换能器空间三维坐标信息。

#### 8. 校准与调试功能

- **固件校准 (Calibration)**：
  - 设备内置了 60 个阵元的初始时间延迟校准数组（单位：微秒），用于补偿硬件走线、换能器个体差异等造成的本底相位误差。
  - 可通过按键 (`KEY0`) 或命令进入校准模式，在校准模式下所有通道相位清零。

- **板载指示灯 (LEDs)**：
  - 通过 `GPIOA` 的引脚控制。
  - 包含 1Hz 心跳灯 (`LED0`)、校准模式指示灯 (`LED1`) 以及 Demo 模式状态指示灯 (`LED2`)。



### 实验1

#### 实验1设计

由于每种调制刺激模式产生的触觉感知绝对值较弱，导致相对的对比差异也较弱。为了保证能够获取到足够的正确的对比差异，实验设计采用对比式实验范式。

1. 实验阵列与组合
共有5个自变量（$DLM_2$, $DLM_3$, $ULM_L$, $LM_L$, $LM_C$）。
将它们进行两两组合，总共产生 $C(5,2) = 10$ 个比较对（例如：$DLM_2$ vs $LM_C$）。
考虑到呈现顺序的遗留效应（先出现的刺激会被记忆衰减影响），需要引入正反序。每个Block包含20个试次（Trials）（即10个比较对 $\times$ 2次呈现顺序）。

2. 硬件与物理环境控制
听觉遮蔽：被试必须佩戴播放固定的白噪声的降噪耳机，阻断相控阵开关时产生的轻微声波或高频噪音泄漏导致对被试触觉感知判断失误。
固定与耦合：被试的手掌必须放置在一个固定高度的泡沫垫上，控制手掌到超声换能器间的距离和位置固定。

##### 实验流程（按被试进行）

使用 G*Power 计算实验1需要的样本大小，设置（Effect size g=0.2, alpha err prob=0.5, Power=0.8，Constant proportion = 0.5），计算出的Total sample size = 49，由于每个被试需要进行正反两次呈现顺序，即每个被试会对同一个比较对贡献 2 个 Trials。因此，计算出所需被试数为 $49 / 2 = 25$ 人。

最终招募 30 名 被试进行实验1。性别比例为 1:1（male:female = 13:13）。

实验分为两个独立的Block（模块），分别为“强度比较”模块和“空间精度比较”模块。两个模块的呈现顺序对每个被试进行随机化平衡（例如一半被试先做强度模块，另一半先做精度模块），以消除顺序效应。

- **Block设计与随机化**
  - **Block 1（强度/精度）**：包含20个Trials（10个刺激对组合 $\times$ 2次呈现顺序）。
  - **Block 2（精度/强度）**：重新随机化呈现这20个Trials。
  - **双盲控制**：实验者与被试在实验进行中均不知道当前呈现的是哪两个调制刺激条件。

- **单次试次呈现（Trial Procedure）**
    呈现“刺激A”（持续1.5秒） $\rightarrow$ 停止（持续1.0秒，用于感受野的机械和神经恢复） $\rightarrow$ 呈现“刺激B”（持续1.5秒）。不允许重复播放刺激，让被试通过第一直觉感受进行选择。

- **强制回答任务（2-AFC）**
    被试根据当前Block的任务要求，做出唯一的强制选择（不允许回答“一样”或“不确定”）：
  - **强度模块（Intensity Block）**：“刚才的两个刺激，哪一个的力度/压力更强？A还是B？”
  - **精度模块（Spatial Precision Block）**：“刚才的两个刺激，哪一个的触觉面积更小、更像是一个清晰的点？A还是B？”

#### 实验1参数设置

所有参数均由刺激频率200Hz、皮肤表面波传播波速以5m/s计算。

所有调制方法的轨迹中心点都为 (0, 0, 0.1) m

- DLM_2：选择的圆环半径为 0.625cm
- DLM_3：选择的圆环半径为 0.481cm
- ULM_L：选择的轨迹长度为 3cm (M=1.2)
- LM_L：选择的起终点距离为 1.5cm，每个周期内轨迹总路程为 3cm (M=1.2)
- LM_C：选择的圆环半径为 0.477cm (M=1.2)

##### 数据处理与理论验证

本研究采用基于贝叶斯推断的混合效应 Bradley-Terry 模型（Bayesian Mixed-Effects Bradley-Terry Model）对成对比较数据进行分析，以评估不同调制方式的主观强度和空间清晰度。

1. **数据预处理**
    - **数据整合**：读取所有被试的实验记录文件（CSV格式），剔除无效试次。
    - **格式转换**：将数据转换为**长格式（Long Format）**。每一行代表一次独立的成对比较（Trial），包含：被试ID（ParticipantID）、刺激A（StimulusA）、刺激B（StimulusB）以及二元选择结果（Outcome，1代表A优于B，0代表B优于A）。
    - **胜率矩阵（Win Matrix）**：生成 $5 \times 5$ 的胜率矩阵，直观展示各调制方式之间的胜负关系。

2. **统计模型构建**
    - **模型框架**：使用基于广义线性混合模型（GLMM）框架的 Bradley-Terry 模型。具体采用 **二项贝叶斯混合 GLM (Binomial Bayes Mixed GLM)** 进行参数估计。
    - **固定效应 (Fixed Effects)**：将5种调制方式（$DLM_2, DLM_3, ULM_L, LM_L, LM_C$）作为固定效应，用于估计每种方式的相对偏好得分（Log-odds Score）。
    - **随机效应 (Random Effects)**：将被试ID（ParticipantID）作为随机效应（Random Intercept），以解释不同被试在感知基准或判断标准上的个体差异（Subject Heterogeneity）。
    - **评分标准化**：模型输出的原始系数（Coefficients）经过中心化处理（Zero-centered），使得所有方法的平均得分为 0，便于直观比较相对优劣。

3. **统计检验与推断**
    - **成对比较 (Pairwise Comparisons)**：基于模型参数的协方差矩阵（Covariance Matrix），使用 **Wald 检验 (Wald Test)** 计算任意两种调制方式之间评分差异的 Z分数（Z-score）和 p值（p-value）。
    - **显著性水平**：
        - `***` : p < 0.001
        - `**` : p < 0.01
        - `*` : p < 0.05
        - `n.s.`: 无显著差异 (p ≥ 0.05)

4. **决策时间分析 (Decision Time Analysis)**
    - **目的**：评估不同刺激对之间的辨别难度（Discriminability）。较长的反应时通常意味着两个刺激的主观差异较小，难以区分。
    - **预处理**：
        - **无效数据剔除**：剔除过短（< 200ms）或过长（> 10s）的异常反应时数据。
        - **对数变换 (Log-transformation)**：对原始反应时进行自然对数变换，以校正反应时数据的偏态分布，使其更符合正态假设。
        - **Z-score 标准化 (Z-score Standardization)**：对每个被试的对数反应时（Log-RT）进行标准化处理（减去均值除以标准差），以消除不同被试基础反应速度的个体差异（Baseline Differences）。
    - **指标计算**：计算每对刺激组合的平均标准化对数反应时（Mean Z-Log-RT），构建反应时矩阵。

5. **结果可视化**
    - **相对评分条形图**：展示各调制方式在“主观强度”和“空间清晰度”上的相对表现（Log-odds 尺度）。
        - **误差棒**：展示 **95% 置信区间 (95% CI)**，计算公式为 $\pm 1.96 \times SE$。
        - **显著性标注**：使用连接线和星号标注具有统计学显著性差异的对比对。
    - **决策时间热力图**：使用热力图（Heatmap）展示各刺激对之间的平均标准化决策时间（Z-Log-RT）。颜色越深代表决策时间越长，区分难度越大。


#### 实验1结果

### 仿真1

#### 仿真1设计

本仿真旨在通过数值模拟验证实验1中涉及的五种触觉调制方式在软组织介质中的波动传播特性。仿真基于 **MATLAB R2025b** 平台及 **k-Wave** 声学仿真工具箱（`pstdElastic3D` 求解器），利用高性能工作站进行大规模并行计算。

**1. 仿真环境与物理模型**

- **计算域**：模拟尺寸为 $80 \times 80 \times 40$ mm³ 的三维空间，网格精度为 $1.0$ mm ($dx=dy=dz=1.0$ mm)。
- **介质参数**：模拟人体皮肤软组织。
  - 密度 ($\rho$): $923.5$ kg/m³。
  - 声速：剪切波速 ($c_s$) 设为 $5.0$ m/s；纵波波速 ($c_p$) 物理值为 $1540$ m/s，为提高计算效率，仿真中设定为 $100$ m/s (满足 $c_p \gg c_s$ 且不影响剪切波主导的触觉感知特性)。
  - 衰减：剪切波衰减系数 $\alpha_s = 10$ dB/(MHz$^y$ cm), $y=2.0$；纵波衰减系数 $\alpha_p = 0.1$ dB/(MHz$^y$ cm), $y=1.5$。
- **边界条件**：四周及底部设置 PML (Perfectly Matched Layer) 吸收边界，并配合 20 层 Sponge Layer 以消除边界反射，模拟半无限大空间。
- **时间步长**：根据 CFL 数 ($0.3$) 自动计算，确保数值稳定性。

**2. 激励源设置 (Excitation Source)**

- **源类型 (Source Type)**: 垂直应力源 ($S_{zz}$)，模拟聚焦超声产生的恒定声辐射力 (Constant Radiation Force)。
- **时间特性 (Temporal Profile)**: 源幅度在时间上保持恒定 (DC)，通过空间位置的变化产生调制效果。
- **扫描频率 (Scan Frequency)**: $f_{mod} = 200$ Hz (周期 $T=5$ ms)。
- **持续时间 (Duration)**: $75$ ms (15个扫描周期)。
- **空间分布 (Spatial Profile)**: 高斯分布点源 ($Gaussian$)。
  - **分布宽度**: 标准差 $\sigma = 4.25$ mm (代码参数 `ring_width`)。
- **强度 (Intensity)**: 峰值应力幅度设为 $100$ Pa。

**3. 调制方式与参数配置 (Modulation Methods & Configurations)**
本仿真对比了五种焦点扫描策略。针对每种模式，具体的几何参数与运动学参数设置如下（剪切波速 $c_s = 5.0$ m/s）：

1. **DLM_2 (Discrete Lateral Modulation - 2 Points)**: 焦点在圆周上2个离散点间切换，半径 $6.25$ mm。
2. **DLM_3 (Discrete Lateral Modulation - 3 Points)**: 焦点在圆周上3个离散点间切换，半径 $4.81$ mm。
3. **ULM_L (Uniform Lateral Modulation - Line)**: 焦点沿直线单向匀速扫描，长度 $30$ mm。(M=1.2)
4. **LM_L (Lateral Modulation - Line)**: 焦点沿直线双向往复扫描，长度 $15$ mm。(M=1.2)
5. **LM_C (Lateral Modulation - Circle)**: 焦点沿圆周连续匀速扫描，半径 $4.77$ mm。(M=1.2)

**4. 数据采集与分析**

- **观测平面**：记录介质表面下方 $1$ mm 深度的水平切面数据。
- **记录物理量**：应力张量分量 ($\tau_{xy}, \tau_{xz}, \tau_{yz}$) 及声压 ($p$)。
- **分析指标**：
  - **RMS Stress Map**: 稳态期间的应力均方根值分布，评估能量分布。
  - **Peak Stress Map**: 最大应力峰值分布。
  - **Rate of Change (Jerk) Map**: 应力随时间变化率的最大值，反映触觉刺激的动态变化强度。
  - **Spatial Gradient**: 应力场的空间梯度模，反映触觉的空间分辨率线索。
  - **Spatiotemporal Slice**: 提取中心截面的时空图 (XT-slice)，分析波动的传播与干涉模式。

**5. 可视化与数据后处理 (Visualization & Post-processing)**

为确保不同调制方式之间的公平对比及波场特性的准确呈现，可视化过程中采用了以下关键处理方法：

- **全局归一化 (Global Normalization)**: 对所有对比组（5种调制方式）的同类物理量（如RMS应力、峰值应力）采用统一的色标范围（基于所有组中的全局最大值），以直观反映相对强度差异。
- **频谱分析预处理**: 在绘制中心点波形的频谱图之前，对时域信号进行了 **去均值 (Demeaning)** 处理以消除直流分量，并施加 **汉宁窗 (Hann Window)** 以抑制频谱泄漏 (Spectral Leakage)，随后进行 FFT 变换并转换为单边频谱 (Single-Sided Spectrum) 的 dB 刻度显示。
- **时空图 (XT-Diagram) 增强**: 在绘制时空切片时，对每一空间点的时域信号进行了去均值处理，去除了静态背景应力，从而高亮显示动态波动的传播轨迹与干涉条纹。
- **波前快照 (Wavefront Snapshots)**: 提取首个周期内四个相位点 ($T/4, T/2, 3T/4, T$) 的瞬时波场，同样采用去均值处理以突出波动的正负极性（压缩/拉伸），使用发散色图 (Diverging Colormap) 清晰呈现波前的空间结构。
- **聚焦性能评估**: 提取焦平面上的峰值应力分布曲线，进行归一化处理，并计算 **半峰全宽 (FWHM)** 作为空间聚焦分辨率的量化指标。

**6. 计算平台 (Computing Platform)**

- **操作系统**: Ubuntu 24.04.3 LTS (Noble Numbat)
- **处理器 (CPU)**: AMD Ryzen 9 9900X (12-Core, 24-Thread, up to 5.6 GHz)
- **内存 (RAM)**: 64 GB
- **图形加速 (GPU)**: NVIDIA RTX5080 16GB GPU (Driver Version: 580.126.09)

#### 仿真1结果

以下分析均基于表面下 **1 mm** 深度切面的应力场结果。由于仿真将纵波声速压缩为 100 m/s，绝对应力量级主要用于相对比较，空间形态、频谱结构、传播模式的相对差异具有核心物理解释价值。

##### `comparison_spatial_focusing` 仿真结果

- **横向聚焦宽度 (FWHM) 排序与形态**：DLM_3 (25 mm) < LM_C (26 mm) < LM_L (28 mm) < DLM_2 (33 mm) < ULM_L (42 mm)。所有曲线均呈现对称双峰与中心低谷形态，表明被强化的并非几何中心，而是中心两侧的剪切应力带。
- **模态演化特征**：DLM_3 与 LM_C 的曲线几乎重合，具有最窄的中心谷和最陡的上升边缘，表明三点离散切换 (DLM_3) 在空间聚焦上已近似实现连续圆周 (LM_C) 的局域化双叶型旋转剪切场。DLM_2 表现为较宽的中心平台，属于双驻留点干涉区；LM_L 表现为端点增强但不紧致的分布；ULM_L 能量分布最分散且中心残余响应高。
- **仿真与知觉的解耦**：稳态 FWHM 排序（LM_C/DLM_3 最优）与主观实验的空间清晰度排序（ULM_L/DLM_2 最优）存在不一致。这表明稳态横向能量收缩度不足以单独预测主观清晰度，真实感知高度依赖时间变化率、波前扫过时序及动态受体激活。

##### `comparison_cross_section_rms` 仿真结果

- **稳态能量空间分布**：截面 RMS 剪切应力均呈现双峰对称分布，能量沉积在轨迹邻域或干涉带而非几何中心。LM_C 和 DLM_3 形成深谷高对比度的轴对称旋转环带；LM_L 形成沿轨迹的宽双叶积累区；DLM_2 表现为典型的双驻留点外侧能量分布；ULM_L 空间轮廓最平缓。
- **峰值强度排序**：LM_L ≈ DLM_3 (约 19.5–20 kPa) > LM_C (约 18.5–19 kPa) > DLM_2 (约 16.5–17 kPa) > ULM_L (约 10.5 kPa)。
- **稳态能量与主观强度的背离**：仿真中稳态 RMS 最低的 ULM_L 在实验中主观强度最高，而 RMS 较高的调制方式表现相对较弱。这明确证实触觉强度不等于稳态 RMS 剪切能量，波前压缩机制与传播组织度在感知中起决定性作用。

##### `comparison_spectrum` 仿真结果

- **基频保真度**：在 200 Hz 基频处，DLM_2、ULM_L 和 LM_C 展现出最高峰值（45–48 dB），表明单向线扫、两点切换和圆周连续扫描能有效维持主调制频率。
- **倍频漂移现象**：LM_L 在 200 Hz 处能量极低，而在 400 Hz 处出现最强峰（约 40 dB）。这证实了双向往复扫描由于周期内重复扫过中心区域，天然引入了 2 倍频成分，导致能量脱离机械感受器的最佳敏感区。
- **频谱集中性**：ULM_L 的能量高度集中在受体敏感的 200 Hz 区间，验证了单向扫描避免倍频的物理初衷；DLM_3 则在 200 Hz 至 1000 Hz 存在多个显著的谐波峰，反映了多点离散切换引入的高时间复杂度。

##### `RMS/Peak` 仿真结果

- **应力场拓扑结构**：五种调制的极值均未出现在几何中心，证明触觉主导刺激是表面剪切波在轨迹附近形成的时空干涉带。DLM_2 呈现水平双热点结构（点间定时干涉）；DLM_3 收缩为近方圆环带（离散旋转模态）；LM_C 形成最标准的轴对称环形分布；LM_L 呈现沿扫描方向展开的双叶结构。
- **离散到连续的演进**：DLM_2 到 DLM_3 的转变显示了波场从“双驻留点交替”向“离散旋转场”的物理跃迁，进一步支撑了 DLM_3 逼近 LM_C 稳态特性的结论。

##### `Signed XT Diagram` 仿真结果

- **横向剪切基本模式**：所有调制均呈现上下异号、中心近零的模式，几何中心是高梯度的符号翻转带而非幅值峰区，证实触觉线索源于剪切偶极/旋转型应变结构。
- **动力学指纹特征**：
  - **LM_C**：具有最稳定的平行双带反对称结构与固定的零交叉线，代表纯粹的连续旋转稳态。
  - **LM_L**：主带存在明显的形态转折与半周期不对称增强，刻画了方向翻转诱发的相位重置与端点换向瞬态。
  - **ULM_L**：呈现多条具明确斜率（~5 m/s）的连续斜向条纹，提供了表面剪切波持续单向传播与波前压缩的最直接证据。
  - **DLM_2**：表现为明显的块状驻留与交替重构，印证其动力学本质为离散切换的局域驻留。
  - **DLM_3**：表现为紧凑的三相循环重组结构，是介于离散切换与连续旋转之间的过渡模态。

##### `Max Rate of Change` 仿真结果

最大应力变化率（$|d\tau/dt|$，即 Jerk Map）反映了触觉刺激的动态突变强度。由于人体皮肤中的快速适应性机械感受器（如 Pacinian 小体）对刺激的时间变化率高度敏感，该指标是评估触觉激活效率的关键维度。

- **离散切换导致的中心极值突变 (DLM 系列)**：
  - **DLM_2** 呈现出明显的“双驻留点 + 中心极核”结构。在两个焦点切换的几何中心，存在一个极高变化率的“红点”（接近 $1.4 \times 10^6$ Pa/s）。这证明了离散焦点的瞬间空间跳跃（相当于无限大的移动速度）会在波前交汇处引发剧烈的时空剪切突变，从而强效激活感受器。
  - **DLM_3** 同样在中心保留了极高变化率的尖峰，但外围的高变化率区域收缩为紧凑的螺旋/三相交织形态。这表明三点切换在制造强烈局部瞬态刺激的同时，波场正向旋转连续性过渡。
- **连续运动的平滑与区域化分布 (LM 系列)**：
  - **LM_C** 形成了一个完美的轴对称圆环形高变化率带，中心区域反而表现为低值“空洞”。这反映了焦点连续圆周运动产生的平滑旋转剪切场：变化率的峰值稳定地分布在焦点轨迹附近，避免了类似 DLM 的中心剧烈突变。
  - **LM_L** 呈现出沿 X 轴拉长的双叶分布，在轨迹的两端出现了较强的变化率热点（橙/黄色环）。这是由于焦点在直线两端发生 $180^\circ$ 速度翻转时，新波前与尾迹低压区碰撞产生的相位重置与瞬态冲击。
- **单向持续压缩的温和分布 (ULM_L)**：
  - **ULM_L** 的变化率场呈现出明显的不对称“彗星”拖尾结构，且整体峰值是五种模式中最低的（缺乏红色极值区）。左侧（焦点起始/压缩端）形成了一个黄橙色较强区域，随后向右侧逐渐耗散。
- **仿真与知觉机制的深层解耦**：
  - 这是一个非常有趣的物理与心理学背离现象：在心理物理实验中，**主观强度得分最高的是 ULM_L，但在本仿真中，ULM_L 的最大变化率（以及之前的稳态 RMS 能量）却是最低的/最分散的**；相反，DLM_2 具有极高的局部变化率突变。
  - 这一反差深刻地揭示了：**最强的超声触觉感知并非来源于瞬间、局部的极值暴力撕扯（高 $|d\tau/dt|$），而是来源于 ULM_L 这种通过速度匹配实现的、对波前长达整个周期的“持续定向压缩与累积”**。触觉系统在宏观上更倾向于对具备明确方向性和时空连贯性的剪切波前积分，而非单一的局部突变。

##### `Gradient Snapshot` 仿真结果

应力空间梯度（$|\nabla \tau|$，单位 Pa/m）反映了介质内部相邻微元之间剪切力的空间差异程度。在经典的触觉感知理论中，较高的空间梯度通常被认为能更有效地激发机械感受器（产生强烈的拉扯感）并提供清晰的空间边界。然而，本仿真揭示了更复杂的机理。

- **中心汇聚与极值突变 (LM_C, DLM_3, DLM_2)**：
  - **LM_C** 和 **DLM_3** 的梯度场呈现出高度相似的拓扑结构：外围形成近似圆形的边界环，而几何中心出现了一个极度狭窄且强度极高（呈现暗红色，接近 $6.5 \times 10^6$ Pa/m）的梯度极点。这表明无论是连续圆周还是三点离散切换，波前向心传播干涉都会在圆心引发剧烈的空间剪切差。
  - **DLM_2** 在中心区域形成了一条由三个离散高梯度热点组成的虚线带，反映了两个焦点交替跳跃引发的非对称空间撕扯。
- **驻波与行波的梯度拉伸 (LM_L, ULM_L)**：
  - **LM_L** 在中心呈现出垂直于扫描方向的短条状高梯度区（橙红色），这是焦点双向往复扫过并在中心叠加的结果。
  - **ULM_L** 的梯度分布呈现出最显著的异质性。它沿扫描方向拉长为宽阔的水平带，且**绝对峰值极低**（最高仅为黄绿色，缺乏红色极值区）。波前沿单一方向持续推移，使得空间应力差被平缓地摊薄在更长的轨迹上。
- **关键的有趣发现（物理与心理物理学的反直觉解耦）**：
  - 结合实验1的结果，此处出现了一个极其关键且违背直觉的现象：在主观评价中**强度最高且空间清晰度最好**的 **ULM_L**，其空间梯度峰值却是五种模式中**最低且分布最分散**的。相反，中心具有极高梯度的 LM_C 和 DLM_3 表现平平。
  - **理论推论**：这一结果基于第一性原理有力地证明了，在超声触觉（微弱的表面剪切波环境）中，**人类触觉感知的最优化并非由“局部瞬态的空间撕扯力（绝对高梯度）”决定，而是由“时空积分的连贯性”决定**。ULM_L 虽然空间梯度平缓，但其速度匹配了表面波速（马赫锥压缩），为感受器提供了沿单一方向**长时间、无破坏性重置**的连续波前累加。相比之下，单纯制造极高的中心空间梯度（如 LM_C），反而可能因为缺乏宏观的定向波前推移，导致神经发放缺乏一致的相干性。


##### `Center Waveform` 仿真结果

中心点时域波形（Center Waveform）提取了几何中心处的应力随时间变化的动态过程（展示了 60~75 ms 内共 3 个调制周期的稳态数据）。该结果直接揭示了不同空间调制轨迹在固定空间点上引发的**局部频率突变（Frequency Shifting）**与**谐波特征**，为实验1中的心理物理学和声学现象提供了决定性的底层物理证据。

- **LM_L 的绝对倍频效应 (Frequency Doubling)**：
  - **现象**：`LM_L` 的波形在每个 5 ms 的周期（200 Hz）内，呈现出两个完全对称且等高的波峰（峰值约 450-480 Pa），即中心点实际承受的是完美的 **400 Hz** 物理应力刺激。
  - **物理与生理学解释**：焦点在直线上双向往复运动时，每个周期内必定两次扫过中心点。这一绝对的倍频效应直接将刺激主频从机械感受器（Pacinian小体）最敏感的 200 Hz 区间移出，这是导致 `LM_L` 在主观强度实验中得分垫底的根本物理原因。

- **ULM_L 的基频保真与不对称压缩**：
  - **现象**：`ULM_L` 波形在每个周期内仅存在一个主峰（峰值约 780 Pa），且波形呈现高度不对称性（陡峭的上升沿与带有次级波动的缓降沿）。
  - **物理与生理学解释**：单向直线扫描成功避免了 `LM_L` 的倍频缺陷，将能量严格锁定在 200 Hz 基频。不对称的波形轮廓是波前单向压缩（马赫锥）扫过中心点的典型时域特征。它在维持最佳受体响应频率的同时，提供了连续的定向波前，确立了其作为**最优触觉调制方案**的地位。

- **LM_C 的极高频谱纯度**：
  - **现象**：`LM_C` 展现出极其平滑、近乎完美的正弦波形，单周期单峰（峰值约 700 Pa）。
  - **物理与生理学解释**：平滑的圆周连续运动在中心点激发出纯净的 200 Hz 旋转剪切场，没有任何高频瞬态成分。这解释了其实验中“极低可听噪声”的现象。

- **DLM_3 与 DLM_2 的离散谐波注入**：
  - **现象**：`DLM_3` 的波形在单个 5 ms 周期内出现了三个明显的子波峰（峰值分别为约 780、580、500 Pa），引入了强烈的 **600 Hz** 结构性频率。`DLM_2` 则呈现单周期内一高（~1180 Pa）一低的不规则双峰。
  - **物理与生理学解释**：离散焦点的空间跳跃必然在局部引入与跳跃次数成正比的谐波频率。`DLM_3` 的三点切换直接导致了中心点受力的 3 倍频（600 Hz）现象。这完美解释了实验1中观察到的“DLM_3 低频噪声频率明显比 DLM_2 更高”的声学现象，并指出由于 600 Hz 远超触觉受体敏感区，其能量利用率发生衰减。

**综合结论**：
波形分析确凿地证明，空间调制轨迹的设计不仅改变能量的空间分布，更会从根本上重塑局部介质的**时间频率结构**。决定触觉强度的核心要素并非中心点的绝对物理应力峰值（如 `DLM_2` 虽具有 1180 Pa 的最高中心峰值，但主观感知并非最强），而是**刺激主频与受体敏感频段（200 Hz）的对齐程度**。单向连续调制（`ULM_L`）是唯一在避免倍频漂移的同时实现波前压缩的最优干涉策略。

##### `Wavefront Snapshots (tau_xy & uz)` 仿真结果

波前快照（Wavefront Snapshots）提取了单周期内四个关键相位点（$t= T/4, T/2, 3T/4, T$）的瞬时空间波场分布。其中，质点垂直速度（$u_z$）直观地反映了超声声辐射力焦点的瞬时空间位置与形态；而横向剪切应力（$\tau_{xy}$）则刻画了软组织内激发的表面剪切波的传播轨迹与干涉极性（红蓝交替代表拉伸与压缩）。通过去均值处理（Demeaning），这些快照清晰地揭示了不同焦点运动学模式如何塑造表面波的宏观时空结构。

- **离散点源的时空交织 (DLM_2, DLM_3)**：
  - **现象**：在 $u_z$ 快照中，**DLM_2** 和 **DLM_3** 表现为焦点在不同空间位置的瞬间“闪现”而非平滑移动。对应的 $\tau_{xy}$ 快照显示，每次焦点驻留都会在局部激发出独立的、向外扩散的四极子（Quadrupole）剪切波纹。随着时间推移，这些独立生成的波前在几何中心区域发生碰撞和干涉。
  - **物理推论**：这种离散切换打断了波前的连续性。虽然在驻留点及其交汇处（如 DLM_2 的中心）会产生强烈的瞬态干涉（对应极高的 $|d\tau/dt|$），但未能形成统一的宏观行波，这也是其在触觉感知中带有“粗糙感”或次优表现的物理根源。

- **完美旋转剪切场的形成 (LM_C)**：
  - **现象**：**LM_C** 的 $u_z$ 呈现出平滑的圆周轨迹。其 $\tau_{xy}$ 演化最为独特，形成了一个极其平滑且首尾相接的“螺旋/太极”形（Swirl）剪切应力场。红蓝应变区随着焦点的移动在空间中连续旋转。
  - **物理推论**：连续的圆周运动在中心区域构建了一个无间断、无突变的纯粹旋转剪切场。这种极度平滑的波前组织形式解释了其在时域上的极高频谱纯度（无高频谐波）以及实验中极低的可听噪声，但也因为缺乏定向的波前挤压，导致其主观强度得分偏低。

- **马赫锥压缩与方向重置 (ULM_L vs LM_L)**：
  - **现象**：在 **ULM_L** 中，$u_z$ 稳定地从左向右平移。其 $\tau_{xy}$ 快照极其清晰地展示了“马赫锥（Mach Cone）”效应：由于焦点移动速度（$M=1.2$，超剪切波速），波前在焦点运动的前方（右侧）被持续压缩，形成一个锐利的“V”型（或月牙形）高强度波前，并在整个周期内维持单一方向的推进。
  - **相反的 LM_L**：**LM_L** 在 $t=T/4$ 到 $t=3T/4$ 期间表现出与 ULM_L 类似的前向压缩；但在 $t=3T/4$ 时（焦点到达端点），焦点速度发生 $180^\circ$ 反转。此时新生成的波前与前半周期留下的尾迹发生剧烈的逆向碰撞，导致波前结构的破裂和相位的强制重置。
  - **物理与心理物理学映射**：**ULM_L** 快照直观证明了它是唯一能够实现**“全周期、单向、无破坏性持续波前压缩”**的调制方式。这种将机械能量沿单一空间方向进行连贯时空积分的机制，完美解释了为何其产生的空间梯度虽然平缓，却能激发出**最强且最清晰**的触觉感知。相比之下，LM_L 的方向翻转不仅引入了破坏性干涉，还导致了致命的倍频效应。



### 结合实验1与仿真1的综合分析

- **DLM_3** 与 **LM_C** 对比，各项实验结果和仿真结果均非常接近。在心理物理学实验中两者在强度分数、精度分数上都非常接近，并且被试在区别两者时的反应时长比起其他刺激对显著更长，这个结果也表示两者非常相似，使用三个离散聚焦点“转移”即可实现绝大部分LM_C的效果，工程上可以用来指导刺激调制方式设计。LM_C的优势在聚焦点平滑移动，因此可以实现几乎没有可听频段的噪声。
- 但是DLM_3在两个分数上都略高于LM_C（但不显著），这可能表明由于DLM_3的离散控制，每个聚焦点停留时间都较长，从而可以充分给皮肤足够时间产生响应。而LM_C方法由于是连续聚焦点移动，可能使得皮肤上每个点来不及响应，到不了最大程度的响应，从而影响效果。


### 计算神经动力学模型（Computational Neural Dynamics Modeling）

本研究采用一个**以 Pacinian corpuscle（PC，FAII）为主导通道的、具有机械约束的时空编码模型**，用于在 **k-Wave 物理场仿真** 与 **心理物理学主观评分** 之间建立可检验的神经生理桥梁。该模型的核心任务不是精确重建单个受体的全部微观生物力学细节，而是用**最小但机制上正确**的外周编码框架，回答本研究的两个核心现象：

1. **为什么 ULM_L 在主观上同时表现出更高的强度与更高的空间清晰度；**
2. **为什么 LM_L 会出现显著的“倍频崩溃”（frequency-doubling collapse）。**

---

本研究采用如下方案。

#### 模型定位与理论依据

该模型本质上是一个 **PC-dominant spatiotemporal bandpass-LIF model**，由三层机制构成：

- **机械层**：PC 的层板结构天然形成带通性质，强烈抑制静态压力与低频缓变刺激；
- **神经层**：PC 末梢可被快速动态刺激驱动，呈现高时间精度、相位锁定、快速适应的放电行为；
- **群体层**：触觉强度与空间清晰度并不是由某个点的物理极值单独决定，而是由**整个受体群体在时空上的有效脉冲整合**决定。

这一定义与本研究的中心叙事完全一致：  
> **超声触觉感知的决定因素不是局部物理极值本身，而是表面剪切波前的时空连贯性（spatiotemporal coherence）与受体敏感频段的对齐程度（frequency fidelity）。**

---

#### 建模假设

为保证模型既有解释力又可实现，本研究明确采用以下假设：

1. **主导通道假设**：在本研究的 200 Hz 附近调制条件下，主导感知通道为 **PC/FAII 通路**。  
   这不是说 SA1、FA1、SA2 完全不参与，而是说当前模型首先抓住最有决定性的高频动态通道。

2. **有效通道假设**：40 mm × 40 mm ROI 内布置的 320 个节点代表**有效 PC-like 通道**，用于近似群体编码统计。

3. **标量驱动假设**：受体输入由局部动态应力的**等效标量**驱动，而不是单个固定方向分量驱动。

4. **快速适应假设**：PC 对静态偏置不敏感，主要对 100–400 Hz 动态分量响应，在 200–300 Hz 区间最有效。

5. **群体解码假设**：  
   - **强度**由稳态时间窗内的总群体脉冲数编码；  
   - **空间清晰度**由群体脉冲在二维空间上的集中程度编码。

---

#### 模型总览

模型输入为 MATLAB / k-Wave 导出的时空应力场，输出为两个心理物理学对应指标：

- **Intensity Score**：群体总脉冲数
- **Spatial Clarity Score**：二维脉冲密度半峰面积的负对数

完整流程如下：

1. **阶段一：空间感受野拓扑初始化**
2. **阶段二：PC 生物力学带通滤波**
3. **阶段三：机械转导与 LIF 放电**
4. **阶段四：群体解码与量化指标输出**

---

#### 阶段一：空间感受野拓扑（Topography Initialization）

##### 阶段一：ROI 与有效受体阵列

- ROI 尺寸：`40 mm × 40 mm`
- 受体通道数：`N = 320`
- 节点分布：采用**固定随机种子下的 Poisson-disk 采样**
- 最小间距：`d_min = 1.8 mm`

采用 Poisson-disk 采样而不是完全均匀网格，有两个原因：

1. 它比纯随机分布更稳定，不会出现非生理性局部密集簇；
2. 它比完全规则网格更接近真实受体的非晶格拓扑。

> **实现建议**：固定 `random_state = 2025`，保证所有条件共用完全相同的受体拓扑，避免拓扑差异引入额外方差。

---

##### 阶段一：输入物理量定义：等效动态应力（剪切模长）

本研究中，受体驱动定义为**剪切应力分量的欧氏模长**：

\[
S_{\mathrm{eq}}(x,y,t) = \sqrt{\tau_{xy}(x,y,t)^2 + \tau_{xz}(x,y,t)^2 + \tau_{yz}(x,y,t)^2}
\]

其中 \(\tau_{xy}, \tau_{xz}, \tau_{yz}\) 为仿真输出的切应力分量。该定义的特点是：

- 综合了 xy, xz, yz 三个平面的剪切作用；
- 比单独使用 \(\tau_{xy}\) 更全面、更接近真实受体所受机械驱动；
- 注意：此处未包含法向应力分量（因仿真输出限制未导出完整张量），故并非完整的 Von Mises 等效应力。

---

##### 阶段一：受体空间池化

PC 不是数学上的点传感器，因此每个受体的输入不是单点取样，而是对周围局部机械场做一个高斯空间池化：

\[
S_i(t)=\iint_{\Omega} G_{\sigma_r}(x-x_i, y-y_i)\, S_{\mathrm{eq}}(x,y,t)\,dx\,dy
\]

其中：

- \((x_i, y_i)\) 为第 \(i\) 个受体坐标；
- \(G_{\sigma_r}\) 为二维高斯核；
- \(\sigma_r = 2.0\ \text{mm}\)。

离散实现上，先在每一帧对 \(S_{\mathrm{eq}}(x,y,t)\) 做二维高斯平滑，再用双线性插值采样到受体坐标：

\[
S_i(t) = \text{Interp2D}\big(\text{GaussianBlur}(S_{\mathrm{eq}}(\cdot,\cdot,t), \sigma_r),\ x_i, y_i \big)
\]

> 该步骤是整个模型中非常关键的生物学修正。若直接用单点应力值，模型会被局部网格极值主导，错误夸大离散热点的作用。

---

##### 阶段一：Python 实现细节

推荐使用：

- `numpy`
- `scipy.io.loadmat`
- `scipy.ndimage.gaussian_filter`
- `scipy.interpolate.RegularGridInterpolator`

数据结构建议：

- `stress_eq`: shape = `[T, H, W]`
- `pc_xy`: shape = `[N, 2]`
- `S_raw`: shape = `[N, T]`

---

#### 阶段二：PC 生物力学滤波（Biomechanical Filtering）

PC 的层板结构本质上构成一个**高频优先、静态抑制、极高频衰减**的机械滤波器。为了在最小模型中保留这一机制，本研究采用**二阶 Butterworth 带通滤波器**。

---

##### 去均值

先对每个受体的输入去除静态偏置：

\[
\tilde{S}_i(t) = S_i(t) - \frac{1}{T}\int_0^T S_i(\tau)\,d\tau
\]

离散形式：

\[
\tilde{S}_i[n] = S_i[n] - \frac{1}{N_t}\sum_{m=1}^{N_t} S_i[m]
\]

---

##### 重采样

为兼顾滤波精度与放电模拟效率，将受体输入统一重采样到：

- `fs_model = 10 kHz`
- `dt = 0.1 ms`

该采样率足以稳定表示 100–400 Hz 动态分量，同时能精确实现 1 ms 不应期。

---

##### 二阶带通滤波

使用二阶 Butterworth 带通滤波器：

- \(f_{\mathrm{low}} = 100\ \text{Hz}\)
- \(f_{\mathrm{high}} = 400\ \text{Hz}\)

\[
S_{\mathrm{filt},i}(t)=\mathrm{BP}_{100-400}\{\tilde{S}_i(t)\}
\]

离散实现使用 `scipy.signal.butter(..., output="sos")` 与 `scipy.signal.sosfilt`，采用**因果滤波**而不是 `filtfilt`，原因是：

- 该模型需要保留传播时序；
- 空间清晰度与群体同步性依赖真实时延结构；
- 因果滤波更符合机械传导本身的生理方向性。

> 需要强调：`100–400 Hz` 带通本身**不会把 400 Hz 完全“切掉”**，它只会对高边缘频率产生部分衰减。  
> 因此 **LM_L 的崩溃并不是由“带通滤波单独造成”**，而是由以下三者共同造成：  
>
> 1. 400 Hz 位于 PC 最优敏感区上沿；  
> 2. 双向换向破坏波前时空连贯性；  
> 3. LIF 在有限膜时间常数与不应期约束下，对这种“双次扫过 / 高边缘频率 / 相位重置”的驱动整合效率更低。  
> 这一定义比简单地说“400 Hz 被滤掉了”更符合事实。

---

#### 阶段三：机械转导与放电（Mechanotransduction & Spiking via LIF）

本研究采用**标准漏积分放火（Leaky Integrate-and-Fire, LIF）模型**描述 PC 末梢的神经放电。这里不追求绝对膜电位的生物物理真实性，而采用**规范化参数化**，使不同刺激条件之间的相对比较最清晰、最稳定。

---

##### 驱动电流

PC 仅对带通后的动态正向机械驱动产生有效去极化电流：

\[
I_{\mathrm{stim},i}(t)=\gamma \cdot \max\big(0, S_{\mathrm{filt},i}(t)\big)
\]

其中：

- \(\gamma\) 为全局耦合增益；
- 所有受体共用同一个 \(\gamma\)；
- 本研究只允许**一个全局增益参数**，禁止为不同刺激条件分别拟合。

---

##### LIF 微分方程

\[
\tau_m \frac{dV_i(t)}{dt} = -\big(V_i(t)-V_{rest}\big) + R_m I_{\mathrm{stim},i}(t)
\]

本研究使用如下规范化参数：

| 参数 | 符号 | 数值 |
|---|---:|---:|
| 膜时间常数 | \(\tau_m\) | 2.0 ms |
| 静息电位 | \(V_{rest}\) | 0 |
| 重置电位 | \(V_{reset}\) | 0 |
| 放电阈值 | \(V_{thresh}\) | 1 |
| 膜电阻 | \(R_m\) | 1 |
| 绝对不应期 | \(t_{ref}\) | 1.0 ms |

离散更新公式：

\[
V_i[n+1] = V_i[n] + \frac{\Delta t}{\tau_m}\Big( -(V_i[n]-V_{rest}) + R_m I_{\mathrm{stim},i}[n] \Big)
\]

发放规则：

- 当 \(V_i[n] \ge V_{thresh}\) 时，记录一个 spike；
- 随后令 \(V_i \leftarrow V_{reset}\)；
- 进入 \(1\ \text{ms}\) 绝对不应期。

---

##### 全局增益 \(\gamma\) 的标定原则

为避免模型过拟合，本研究只进行**一次单点标定**：

> 将 \(\gamma\) 固定为使 **LM_C 条件下，稳态阶段中位数活跃受体平均每个 5 ms 周期产生 1 个 spike**。

理由：

1. LM_C 的时频结构最平滑，适合作为中性基准；
2. 标定后其余四种刺激全部为**外推预测**；
3. 这样可直接检验“频率保真 + 时空连贯性”是否足以解释实验排序。

---

##### Python 实现细节

推荐使用全向量化 NumPy 实现：

- `V`: shape = `[N]`
- `refractory_counter`: shape = `[N]`
- `spike_train`: `bool` array with shape `[N, T_model]`

每一时间步按如下逻辑更新：

1. 对不处于不应期的神经元进行电位积分；
2. 检测阈值穿越；
3. 记录脉冲；
4. 重置膜电位并刷新不应期计数器。

#### 阶段四：群体解码（Population Decoding）

---

##### 强度感知解码：Intensity Score

定义稳态窗口为仿真结束前最后 `50 ms`：

\[
\mathcal{W}=[t_{end}-50\text{ms},\ t_{end}]
\]

第 \(i\) 个受体在窗口内的放电数：

\[
n_i = \sum_{t \in \mathcal{W}} \mathbb{I}\{ \text{spike}_i(t)=1 \}
\]

总群体脉冲数定义为：

\[
\mathrm{IntensityScore} = \sum_{i=1}^{N} n_i
\]

该指标与实验 1 中的 **Intensity Preference** 对应。  
理论上，若“受体敏感频段对齐 + 波前相干积分”假设成立，则：

\[
\mathrm{ULM\_L} > \mathrm{DLM_2} \approx \mathrm{DLM_3} \approx \mathrm{LM_C} > \mathrm{LM_L}
\]

这里最关键的不是局部应力峰值，而是**多少受体能在最优频带内被持续、相干、有效地驱动进入放电**。

---

##### 空间清晰度解码：Spatial Clarity Score

首先统计同一稳态窗口内每个受体的放电数 \(n_i\)，再构造二维 spike density map：

\[
\rho(x,y)=\sum_{i=1}^{N} n_i \cdot G_{\sigma_k}(x-x_i, y-y_i)
\]

其中：

- `density grid = 1 mm`
- \(\sigma_k = 2.0\ \text{mm}\)

随后定义半峰区域：

\[
\Omega_{0.5} = \{(x,y)\ |\ \rho(x,y)\ge 0.5\cdot \rho_{\max}\}
\]

其面积为 \(A_{0.5}\)。  
最终空间清晰度指标定义为：

\[
\mathrm{SpatialClarityScore}
=

-\log\left(\frac{A_{0.5}}{A_{ROI}}\right)
\]

其中 \(A_{ROI}=40\times 40=1600\ \text{mm}^2\)。

该定义有三个优点：

1. **维度统一、可比较；**
2. **对条带状、点状、环状激活都成立；**
3. **比 Gini 系数更直观地对应“触觉核心区有多紧”。**

值越大，表示半峰核心区越小，空间知觉越清晰。

---

##### 补充机制指标：Frequency Fidelity Index

为了直接解释 **ULM_L vs LM_L** 的频率机制，本研究额外定义一个**频率保真指标**，用于检验受体输入是否对齐 200 Hz 主敏感带：

\[
\mathrm{FFI}
=

\frac{
\sum_{i=1}^{N} P_i(180\text{–}220\ \text{Hz})
}{
\sum_{i=1}^{N} P_i(380\text{–}420\ \text{Hz}) + \varepsilon
}
\]

其中 \(P_i(f)\) 为第 \(i\) 个受体滤波前输入的功率谱密度。  
\(\mathrm{FFI}\) 越高，表示神经输入越集中在 200 Hz，而不是漂移到 400 Hz。

该指标不是最终感知分数，但它是解释 LM_L 倍频崩溃的最直接中间证据。

#### 11. Python 实现流程

```text
1. 读取 k-Wave 导出的应力张量场
2. 计算等效应力 S_eq(x, y, t)
3. 对每一帧做二维高斯平滑（σ_r = 2 mm）
4. 在 40×40 mm ROI 上生成 320 个 Poisson-disk 受体坐标
5. 用 RegularGridInterpolator 将平滑后的场映射到每个受体，得到 S_i(t)
6. 对每个受体输入去均值
7. 重采样到 10 kHz
8. 用 2 阶 Butterworth 带通 100–400 Hz 做因果滤波，得到 S_filt,i(t)
9. 用 I_i(t) = γ·max(0, S_filt,i(t)) 构造驱动电流
10. 运行 LIF，得到每个受体的 spike train
11. 统计最后 50 ms 的总 spike count 作为 Intensity Score
12. 统计最后 50 ms 的二维 spike density map
13. 计算半峰面积 A_0.5，并输出 Spatial Clarity Score = -log(A_0.5 / A_ROI)
14. 计算 FFI 作为中间机制证据
```

#### 模型预测：如何解释 ULM_L 的高强度与高清晰度，以及 LM_L 的倍频崩溃

---

##### 为什么 ULM_L 最强

在本模型中，ULM_L 具有三项同时成立的优势：

1. **主频保真**  
   ULM_L 在每个周期内对轨迹上多数位置只扫过一次，因此局部驱动频率仍锁定在 200 Hz 附近，而不是被结构性提升到 400 Hz。

2. **单向波前压缩**  
   焦点以接近表面剪切波传播速度的方式单向推进，使得前方波前连续压缩，形成长时间、同方向、无破坏性重置的机械驱动。

3. **群体时空相干积分**  
   这种单向压缩不会在半周期处被反向换向打断，因此大量受体可在相近相位关系下被持续驱动进入放电，导致：
   - **总 spike count 最大**
   - **spike density 核心区最紧致**

换言之，ULM_L 的优势并不是来自最大的局部物理峰值，而是来自**最有效的群体神经整合**。

---

##### 为什么 LM_L 崩溃

LM_L 的失败不是单一因素导致，而是三重机制叠加的结果：

1. **双向往复导致局部两次扫过**  
   轨迹中部每个周期被扫过两次，主驱动频率自然变为 400 Hz。

2. **400 Hz 落在 PC 最优带宽上沿**  
   它不是完全无效，但已经偏离 200–300 Hz 最佳工作区，带通后的有效驱动强度下降。

3. **端点换向造成相位重置与时空去相干**  
   新波前与前一半周期的尾迹发生碰撞，使群体放电在时序上被打散。  
   最终结果是：
   - 有效 spike count 降低；
   - spike density 在空间上更宽、更散；
   - 主观强度与空间清晰度同时下降。

因此，**LM_L 的“倍频崩溃”本质上是“频率失配 + 时空去相干 + 反向换向破坏”的联合后果**，而不是单纯的“物理峰值不够大”。

---

##### 为什么 DLM_3 与 LM_C 会接近

本模型还自然预测：

- **DLM_3** 与 **LM_C** 的群体输出会比较接近；
- 原因是二者都能在 ROI 中形成一个近似连续的旋转型时空激活模式；
- 差别主要体现在：
  - DLM_3 含有更多离散切换引入的高频谐波；
  - LM_C 更平滑、谐波更少、听觉噪声更低。

这与实验和仿真观察一致。

该模型应按以下方式验证，而不是通过大量自由参数“拟合出正确答案”：

##### 最小标定

- ULM_L 的 `IntensityScore` 最高；
- LM_L 的 `IntensityScore` 最低；
- ULM_L 的 `SpatialClarityScore` 显著高于 LM_L；
- DLM_3 与 LM_C 的模型输出接近。

#### 与前人研究的关系：本模型建立在什么基础上

本模型不是凭空构造，而是直接建立在以下经典认识之上：

1. **Pacinian corpuscle/FAII afferent 是高频振动触觉的主导通道**；  
2. **PC 的层板结构具有机械带通性质**，能强烈抑制静态与低频缓变成分；  
3. **FAII 放电可用简化的阈值积分模型近似描述其相位锁定与快速适应行为**；  
4. **群体脉冲统计是连接物理刺激与主观知觉的必要桥梁**；  
5. **已有如 TouchSim 一类工作证明，基于机械输入驱动外周触觉群体放电，是一条成熟可行的建模路线。**

本研究的创新点不在于重新发明触觉神经元模型，而在于把这一路线首次明确地用于解释：

- **超声焦点扫描引发的表面剪切波干涉**
- **频率保真与波前相干性如何决定感知强度与清晰度**
- **为什么 ULM_L 优于 LM_L**
- **为什么 DLM_3 在感知上逼近 LM_C**

#### 本章节的最终结论

本研究最终采用的计算神经动力学模型是：

> **一个以 PC 为主导、以等效动态应力为输入、以 100–400 Hz 机械带通与 LIF 放电为核心、以总群体脉冲数和二维脉冲密度半峰面积为输出的时空编码模型。**

该模型的价值在于：

- 它足够简单，能稳定实现；
- 它足够机制化，不是纯统计黑箱；
- 它能直接解释本研究最关键的反直觉现象：

> **ULM_L 的优势来自“200 Hz 频率保真 + 单向波前压缩 + 群体相干积分”；**  
> **LM_L 的失败来自“400 Hz 倍频漂移 + 端点换向去相干 + 有限神经整合效率”。**

因此，这个模型应作为本研究中连接**物理仿真、神经机制、心理物理结果**的标准解释框架。
